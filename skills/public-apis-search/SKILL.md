# Public APIs Search

Searchable database of **1,558 free public APIs** across **52 categories**.

## When to Use

✅ **USE this skill when:**

- "Find an API for..."
- "I need an API that does..."
- "Looking for a [category] API"
- "What free APIs are available for..."

## Quick Search Commands

### Search by Keyword
```bash
python3 /a0/skills/public-apis-search/search_apis.py search "weather"
```

### List All Categories
```bash
python3 /a0/skills/public-apis-search/search_apis.py categories
```

### Browse by Category
```bash
python3 /a0/skills/public-apis-search/search_apis.py category "Finance"
```

### Find APIs Without Auth (Free to Use)
```bash
python3 /a0/skills/public-apis-search/search_apis.py noauth
```
