## Memory Writing Tips

When you want the agent to remember something, say:
- "Remember this: [info]"
- "Add to memory: [info]"
- "Save this for later: [info]"

The agent will write to either:
- `MEMORY.md` - Long-term, curated memories
- `memory/YYYY-MM-DD.md` - Daily logs and running context
